# 🧮 Calculator Web App

This is a responsive and interactive **Calculator Web Application** built using HTML, CSS, and JavaScript. It supports both button clicks and keyboard input, and includes a **dark mode / light mode toggle** for better user experience.

---

## 🌟 Features
✔️ Perform addition, subtraction, multiplication, and division  
✔️ Real-time calculation display  
✔️ Keyboard input support ⌨️  
✔️ Dark mode 🌙 and Light mode ☀️ toggle  
✔️ Clean and user-friendly interface  
✔️ Responsive design  

---

- CSS3  
- JavaScript

## 📌 How to Use
1. Open the Live Demo  
2. Use mouse clicks or keyboard to enter values  
3. Perform calculations  
4. Toggle between dark and light mode
